# AI Website Copy Generator for Local Businesses

## Project Overview

This project demonstrates the use of Artificial Intelligence tools such as ChatGPT and Claude to generate professional, conversion-focused website content for a local business. The project showcases prompt engineering techniques for creating website-ready copy including homepage content, product descriptions, and call-to-action sections.

## Business Selected

**Elegance Jewels**

## Business Type

Jewelry Store

## Products

* Earrings
* Rings
* Necklaces
* Anklets

## Project Objective

The objective of this project is to create high-quality website content using structured AI prompts. The generated content is designed to be engaging, persuasive, and suitable for direct use on a business website.

## Tools Used

* ChatGPT
* Claude
* GitHub
* HTML/CSS
* Markdown Documentation

## Prompt Engineering Approach

A structured prompting strategy was used to generate website content. Different prompts were designed for:

1. Homepage content generation
2. Product collection descriptions
3. Call-to-action (CTA) sections

The prompts included business details, target audience information, tone requirements, and content structure to ensure consistent and high-quality outputs.
