# AI Website Copy Generator for Local Businesses

## Project Overview
This project demonstrates the use of AI tools such as ChatGPT to generate professional website copy for a local business.

## Business Selected
Elegance Jewels

## Business Type
Jewellery Store

## Products
- Earrings
- Rings
- Necklaces
- Anklets

## Tools Used
- Claude
- ChatGPT

## Features

- Homepage Content
- Product Collection Descriptions
- Call-To-Action (CTA) Sections
- Contact Page Content

## Objective
To create website-ready content using structured prompts and prompt engineering techniques.
